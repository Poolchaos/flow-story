# GitHub Repository Information

## Repository Details

### Description (short - for GitHub)
```
Interactive 3D data storytelling platform - Transform CSV data into animated 3D visualizations with waypoints and smooth camera transitions. Built with React, TypeScript, Three.js, and Tailwind CSS.
```

### About Section (longer description)
```
FlowStory is a web-based platform for creating animated 3D data stories. Upload CSV files, map columns to 3D axes, choose from multiple visualization templates (bars, particles, spheres), capture camera waypoints with annotations, and create cinematic animations through your data. Perfect for presentations, data journalism, and interactive reports.
```
